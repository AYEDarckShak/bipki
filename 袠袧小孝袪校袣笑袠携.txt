1. Добавлена красивая страница авторизации из Website with Login & Registration Form
2. Создана отдельная страница login.html с современным дизайном
3. Добавлена форма регистрации новых пользователей
4. Настроена автоматическая проверка авторизации
5. После успешного входа пользователь переносится на главную страницу (index.html)
6. При выходе пользователь возвращается на страницу входа

📁 НОВЫЕ ФАЙЛЫ:

• login.html - страница авторизации и регистрации
• login-style.css - стили для страницы входа
• login-script.js - логика авторизации с переходом на главную страницу

🔄 ИЗМЕНЕННЫЕ ФАЙЛЫ:

• app.js - добавлена проверка авторизации при загрузке
• index.html - теперь защищена от неавторизованного доступа

🚀 КАК ЗАПУСТИТЬ:

1. Откройте терминал в папке проекта

2. Установите зависимости:
   npm install

3. Запустите сервер:
   npm start
   
   или:
   node server.js

4. Откройте браузер и перейдите по адресу:
   http://localhost:3000

5. Вы увидите страницу входа. Нажмите кнопку "Вход" в верхнем правом углу

6. Используйте тестовые данные:
   
   Менеджер:
   - Логин: admin
   - Пароль: 1234
   - Роль: Менеджер
   
   Участник:
   - Логин: user
   - Пароль: 1234
   - Роль: Участник

7. После успешного входа вы автоматически попадете на главную страницу!

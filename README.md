# Управление проектами - Стартап Студия

## Описание

Веб-приложение для управления проектами стартап-студии с современной системой авторизации и регистрации.

## Новые возможности

✨ **Обновленная система авторизации:**
- Современный дизайн страницы входа
- Форма регистрации новых пользователей
- Красивые анимации и переходы
- Адаптивный дизайн для всех устройств
- Защита главной страницы от неавторизованного доступа

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск сервера

```bash
npm start
```

Или:

```bash
node server.js
```

### 3. Открытие приложения

Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

Вы автоматически будете перенаправлены на страницу входа.

## Использование

### Вход в систему

1. При первом открытии приложения вы увидите страницу входа
2. Нажмите кнопку "Вход" в верхнем правом углу
3. Введите учетные данные:

**Тестовые учетные данные:**
- **Менеджер:** 
  - Логин: `admin`
  - Пароль: `1234`
  - Роль: Менеджер

- **Участник:**
  - Логин: `user`
  - Пароль: `1234`
  - Роль: Участник

4. После успешного входа вы будете автоматически перенаправлены на главную страницу

### Регистрация нового пользователя

1. На странице входа нажмите "Регистрация"
2. Заполните форму:
   - Логин
   - Email
   - Пароль
   - Подтверждение пароля
3. После успешной регистрации войдите с новыми учетными данными

## Структура файлов

```
├── server.js                 # Серверная часть (Express.js)
├── startup_manager.db        # База данных SQLite
├── package.json              # Зависимости проекта
├── login.html                # 🆕 Страница авторизации и регистрации
├── login-style.css           # 🆕 Стили для страницы входа
├── login-script.js           # 🆕 Логика авторизации
├── index.html                # Главная страница приложения
├── app.js                    # Основная логика приложения
└── style.css                 # Стили главной страницы
```

## Технологии

- **Backend:** Node.js, Express.js
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Дизайн:** Градиенты, анимации, адаптивная верстка
